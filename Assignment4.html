PROGRAMMING LANGUAGES

A programming languae is a formal language designed to communicate to a machine particularly a computer.Programming languages can be used to create programs that control the behaviour of a machine and /or to express algorithms precisely.

c++

C++(pronounced see plus plus) is a programming language that is general purpose,statically typed,free-form, multi-paradigm and compiled. It is regarded as an intermediate level language as it comprises both high-level and low-level language features.

C

C is one of the widely used programming languages of all times.C compliers are available for majority of the computer architectures and operating systems.

Java

Java is a computer programming language that is concurrent, class-based, object oriented and specifically designed to have as few implementation dependencies as possible.It is intended to let application developers "write once, run anywhere"(WORA) that means that the codes that runs on one platform does not need to be recompiled to run on another.

O2 and life

Oxygen is a chemical element with symbol O and atomic number 8. In nature, oxygen is a gas with no color or smell. Oxygen is a very important element because it is a part of the air people breathe and the water people drink. Because of this, oxygen supports life. Many living things (including humans) need oxygen to live and breathe, though it is poisonous to some forms of life. In liquid state of matter, oxygen is light blue.[1][2][3]
